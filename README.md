# Hack the Stupid Hack

> Stupid Hack 2023 - Game Mod

## Local development

clone first

then

```
python3 -m venv venv
source venv/bin/activate
```

then

```pip install -r requirements.txt```

and to run locally

```flask run```


if you want changes to appear after saving, run with:
```FLASK_ENV=development flask run```
